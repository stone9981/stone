1.`jekyll new stone`

2.`cd stone` and `git init .`

3.`git add .gitignore _config.yml _layouts _posts css index.html`

4.`git commit -a -m 'comment'` get local master

5.`git remote add origin https://github.com/stone9981/stone.git`

6. `git fetch origin`

7. `git merge origin/gh-pages` merge remote branch gh-pages to local master

8. `git push origin master:gh-pages` push local master to remote gh-pages
